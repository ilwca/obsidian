# ccomp
dentro deste diretório há anotações de aulas e estudos para atividades, projetos e desenvolvimento de artigos separados por matérias. 
Exemplo:

```
ccomp/
├── compiladores/
├── IA/                      # inteligencia artifical
├── PAA/                     # Projeto e análise de algoritmos
├── SAS/                     # Seguranca e auditoria de sistemas
└── Redes/
```  

cada diretório de matéria será composto por anotações em arquivos `.md` escritos usando o software [obisdian](obsidian.md), feito justamente para anotações e estudos.
