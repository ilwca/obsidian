
- **LIS:**   Longest Increase Sequence
- **LCS:** Longest Comun Sequence

## #LIS 
Dada uma sequencia **A**, uma LIS é a maior subsequencia crescente de A, por exemplo:

A = [3, 5, 2, 8, 4, 7 ]

Neste caso **LIS = 3**, sedo **LIS = [3, 5, 8]**

## #LCS
A **LCS** é a Maior subsequencia comum de dois Arrays, assim como a interseção destes de forma ordenada. Por Exemplo: Dados os arrays X e Y o subconjunto xy é um array subconjunto de X e Y, tal que xy também pode ser visto como uma LIS de X e de Y. Exemplo:

X = [2 ,4, 5, 3, 8, 6]
Y = [2, 3, 5, 8, 7, 9]
**LCS = [2, 5, 8]**
